楽天画像ダウンローダー EL COMMUN

このアプリでできること： 楽天の商品画像をまとめて抽出し、パソコンに一括ダウンロードします。
RMSの商品CSV（dl-normal-item）なら「商品画像パス」列から自動で画像を抽出。 URLリストやページのソースなど、画像URLが含まれるテキストの貼り付けにも対応しています。
1

画像URLを入力する 貼り付け or ファイル読み込み

未入力
💡 画像URL（.jpg / .png / .gif など）だけを自動で探して抽出します。関係ない文字が混ざっていても大丈夫です。
📄

RMSの商品CSV（dl-normal-item）をドロップ

またはクリックして選択。URLが書かれたテキストファイルもOK（Shift-JIS対応）
🔍 商品CSVを検出しました ─ 抽出する画像を選んでください
店舗名（URLの一部）: image.rakuten.co.jp/ここの部分/cabinet/… ─ CSVから自動検出されます
2

画像を確認する ダウンロードする画像を選択

0 / 0 枚を選択中
💡 クリックで選択 / 解除を切り替えできます。表示されない画像（🖼️）もダウンロードは試みます。
3

ターミナルでダウンロードする 保存 → 貼り付け → Enter の3ステップ

💡 楽天のサーバーはブラウザからの直接ダウンロードを許可していないため、 Macに標準搭載の「ターミナル」で実行する方式です。難しい操作はありません。
  1. download_rakuten_images.txt を「ダウンロード」フォルダに保存しました✅（実行コマンドもコピー済み）
  2. ⌘＋スペース →「ターミナル」と入力 → Enter でターミナルを開く
  3. ⌘V で貼り付けて Enter — ダウンロードが始まります
完了すると「ダウンロード」フォルダ内の rakuten_images フォルダに R-Cabinetと同じフォルダ構成で画像が保存され、自動で開きます（Cyberduck等でそのまま再アップロードできます）。
・保存先を選ぶ画面が出た場合は「ダウンロード」フォルダを選んでください
・何度か保存すると download_rakuten_images (1).txt のような名前になります。その場合は古いファイルをゴミ箱に入れてから、もう一度保存してください
