🎯 このアプリでできること

楽天RMSからダウンロードした商品CSV（dl-normal-item / dl-item-cat）を、GoQSystemに登録されている商品だけに絞り込むツールです。 GoQの商品データ（itemdata）の「基本コード」を照合キーにして、対象外の商品行を自動で削除し、そのまま楽天RMSに再アップロードできるCSVを作成します。

📦 itemdata（GoQ）
📋 dl-normal-item
🗂️ dl-item-cat 任意
→
🔍GoQの品番で
絞り込み
→
⬇️ dl-normal-item（絞込み済）
⬇️ dl-item-cat（絞込み済）
  • 1商品が複数行にまたがるデータも正しく処理します
  • 商品名の先頭の装飾文字（＼〜／ や【】※一部除く）を自動で削除します
  • ヘッダー行・列の並び・文字コード（Shift_JIS / UTF-8）は元ファイルのまま維持されます
  • dl-item-cat は省略可能（itemdata＋dl-normal-item の2ファイルだけでも実行できます）
