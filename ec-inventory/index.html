EC棚卸し

自社 / メーカー 振り分け出力 v26
楽天のitemdata・dl-normal-item CSVを取り込み、EC在庫を「自社」「メーカー」に自動振り分けして棚卸し用Excelを作成するアプリです。
未入荷の予約商品はご注意ください
手帳・カレンダーなど、まだ入荷していない予約商品が itemdata CSV に含まれている場合は、棚卸しデータを生成する前に、該当行を itemdata CSV から削除してください。入荷前の在庫数がそのまま棚卸し対象に含まれてしまいます。

i

処理ルール

  • A振り分け：システム連携用SKU番号の接頭辞で判定（ec-monthly 準拠）。メーカー接頭辞（8mw-wh-psm- 等）はメーカー、elco-cal- 等は自社。apj-rft- は取り寄せ（非在庫）のため出力対象外。
  • B商品名：itemdata から抽出し、先頭の装飾（【…】＼…／）と ｜ 以降、HTMLタグを除去。
  • C自社：商品名 / 品番 / 原価 / 在庫数 / 合計（＝原価×在庫数）。原価はマスタから品番で突合。
  • Dメーカー：商品名 / 品番 / 上代 / 掛け率 / 在庫数 / 合計（＝上代×掛け率×在庫数）。上代は dl-normal を連携SKUで突合。連携SKUが無い行（単体商品）は商品管理番号（商品URL）で参照。
